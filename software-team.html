Programming

The Programming team is in charge of all the programming needed for the vehicle to move and recognize obstacles during the competition.

The team is subdivided into three groups, each focused on one of the three major software components.

  • Embedded Systems
  • Software Architecture
  • Machine Vision


Establish the communication between the electronic components, the main computer and sensors, as well as transfer all the other teams’ circuits into PCB design with eagle and manufacture.

We use two main protocols of communication:

  • Asynchronous Receiver/Transmitter (UART)
  • Inter-integrated Circuit (I2C)

We use two micro-controllers:

  • The ATMega328P processor
  • Raspberry Pi board

In the hierarchical tree, the two ATMega processors are connected to each other and one of them is connected to the Pi board all by using UART protocol.

UART, compared to parallel communication, uses only two ports. One port to send information and the other to receive the information being transmitted in bytes and assembled again after the reception.

For other peripherals, for example sensors, attached to the AUV that gather information for the decision-making of the system we are using I2C communication protocol.

Develope all mission logic for the AUV, design and code the artificial intelligence system, and make sure that, if the AUV fails, it does so safely. The Robot Operating System (ROS) is being used to simplify code development.

Develop all the code to recognize patterns and objects with our vision cameras. Our main decision making mechanism is recognizing objects with machine learning through our cameras and processing that information to give desired outputs to our ROS system.

To be able to complete the vision tasks specified in the competence the algorithm needs to perform several image processing operations.

Based on these tasks the following specifications for the design process were identified and taken in consideration:

  • Color detection and segmentation
  • Shape detection
  • Adaptive thresholding
  • Contrast stretching
  • Object segmentation
  • Distance calculation from the object to the vehicle
